# 为什么要做这个项目
主要是学习用的，可以掌握一些开发中的一些优质的设计方法。注：这个项目是复刻版，原作者是“程序员Rock”。

# 这个项目的功能主要有哪些
实现了游戏原版的基本的功能，包括游戏的胜负机制，动画的渲染，植物的攻击以及碰撞检测，随机位置生产阳光，阳光的累加以及消耗机制等等。效果展示可以看“介绍”目录。

# 这个项目是怎么被设计出来的
这个是用纯C语言编写的项目，代码主要集中在main.cpp里面（大概1000行左右），用到了c语言的库easyX，这个库只是在windows上才会有。涉及到的技术点如下：
1、使用了池子技术（阳光池）来避免重复创建以及销毁所带来的开销
2、游戏开发逻辑，比如一些初始化工作单独的封装在init函数内部。
3、C语言的结构化编程方法，结构化编程是对功能的抽象而不是对数据的抽象，那么在修改功能逻辑的时候就比较方便不需要改很多的代码，在修改数据的时候改动的地方就很多了（变量是全局的）。
4、对于鼠标、键盘等事件的处理方法（主要使用easyX提供的工具）
# 如果要使用这个项目要注意什么
1、这个项目是开发环境是visual studio 2017
2、不是跨平台的项目，easyX只有windows上才会有。
3、代码量不大，只是学习项目，无法真正投入使用。
